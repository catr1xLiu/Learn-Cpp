> # Remainder and Exponentiation

---

> [!quote] Concept
> 
> #### Remainder (%)

Result takes sign of first operand:

```cpp
7 % 4    // = 3
-6 % 4   // = -2
6 % -4   // = 2
```

Compare remainder against 0 to handle negative numbers:

```cpp
bool isOdd(int x) { return (x % 2) != 0; }
```

> [!quote] Concept
> 
> #### Exponentiation

```cpp
#include <cmath>
std::pow(3.0, 4.0);  // 3^4 = 81 (floating-point only)
```
